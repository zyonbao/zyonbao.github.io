# Margin, Padding, Border

`Margin`, `Padding` are similar but have some important differences. By applying `Margin` to an item you specify the offset a certain edge of the item should have from itâ€™s closest sibling or parent. With `Padding`on the other hand you specify the offset children should have from a certain edge on the parent. `Border`behaves nearly identically to `Padding` and is only separate from `Padding` to make it easier to implement border effect such as color.
